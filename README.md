# MXTtsEngine
基于百度TtsSDK开发的Tts引擎，可以直接调用Android系统tts接口来进行语音合成。

主要包括三块：

- com.ishare.bdtts包：利用百度Tts SDK来进行tts合成。
- com.ishare.speech包：利用Android系统tts接口来进行tts合成（前提是已经安装了tts引擎）
- com.ishare.ttsengine包：基于百度Tts SDK开发的Tts引擎。
